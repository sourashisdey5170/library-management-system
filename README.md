# Library Management System (Python + Tkinter + MySQL)

This is a GUI-based Library Management System built using Python and Tkinter, connected with a MySQL database.

## Features
- User Login & Registration
- Book Management (Add, View, Update, Delete)
- Member Management
- Issue & Return System
- Fine Calculation

## Technologies Used
- Python
- Tkinter (GUI)
- MySQL

## How to Run
1. Install Python
2. Set up MySQL database
3. Run `DatabaseConnection.py`
4. Run 'User.py'
5. Run 'Main_menu.py'
